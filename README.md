# Full-Stack-CRUD-Application
Build a full stack CRUD application using node.js, express, MySQL, and React JS.

## Datebase Query
<code>
CREATE TABLE products(
id INT(11) PRIMARY KEY AUTO_INCREMENT,
title VARCHAR(200),
price DOUBLE,
createdAt DATE,
updatedAt DATE
)ENGINE=INNODB;
</code>
